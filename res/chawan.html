Welcome to Chawan!

This is the default visual home page. You can change it in your configuration file using the start.visual-mode option.

Following is a list of default keybindings for getting started. (Note: C-{key} means control + key, M-{key} means meta + key. Depending on your environment, the meta key may be called Alt or Escape.)

  • q: quit the browser
  • hjkl, arrow keys: move the cursor
  • J, K (or C-e, C-y): scroll up/down by one row
  • C-l: location bar (to enter a URL, etc.)
  • U: reload page
  • , (comma), . (period): previous/next buffer
  • D: discard current buffer
  • M-y: copy current buffer's URL to clipboard (needs xsel)
  • M-p: go to the URL currently on the clipboard (needs xsel)
  • [, ]: move cursor to the previous/next hyperlink
  • C-d, C-u: scroll up/down by half a page
  • C-f, C-b (or PgDn, PgUp): scroll up/down by an entire page
  • (, ) (or zh, zl): shift screen to the left/right by one cell
  • <, >: shift screen to the left/right by one page
  • /, ?: search (or search backwards)
  • n, N: next/previous match
  • C-z: suspend the browser
  • M-C-c: cancel loading
  • H, M, L: move cursor to the Highest/Middle/Lowest rows
  • zz, z.: center on current line (and move to beginning)
  • zt, z-: center on current line (and move to beginning)
  • zb, zC-m: center on current line (and move to beginning)
  • w, b: move cursor to next/previous word
  • v: toggle page source view
  • 0: cursor to first cell on line
  • ^: cursor to first non-whitespace on line
  • $: cursor to last character on line
  • M-,/M-.: previous/next sibling buffer
  • M-/: parent buffer
  • {number}, then movement: repeat movement n times
    (or move to nth line, depending on command)
  • M-c: input command
  • M-cM-c: command mode (browser console)

Happy browsing!
